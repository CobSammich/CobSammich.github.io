Linux Notes

Systemctl

You can check the status of any systemd service with the following command. For example if we wanted to check if sshd, the OpenSSH server process, is running; one would type:

systemctl status sshd

If we want to start a service (using sshd as an example again), we would type:

systemctl start sshd

And to stop a system service:

systemctl stop sshd

Mounting Filesystem to Current System

Using sshfs, you can mount a filesystem to your linux machine. Editing files then will automatically be synced via ssh.

sudo sshfs user@host:/path/to/dir /mnt/droplet -o auto_cache,reconnect,allow_other

To unmount the filesystem that was mounted, you would then type

sudo umount /mnt/droplet

Note that you don't have to mount the whole remote filesystem. You can specify a certain directory to mount.

Finding Files

The locate command is incredibly useful for finding any file on your system. For example, for me to find where my Beginning C++17 textbook pdf is located on my computer, I would simply input the following:

locate "Beginning C++"
