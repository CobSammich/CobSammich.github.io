Sudoku Solver

This program aims to solve sudoku puzzles. The input to the program is an image of a sudoku puzzle. Thus, there are two major portions of this program:

  1. Extract the given sudoku puzzle information from the image into a relevant data structure so we may solve it.
  2. Write an algorithm to solve the sudoku puzzle.

This page will act as a journal for any references, implementation decisions, etc.

Finding the Sudoku board

Some ideas for picking out and finding the corners of the sudoku board are:

  • Matched filter
  • Optical character recognition (OCR)
