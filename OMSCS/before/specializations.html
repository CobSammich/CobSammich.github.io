OMSCS Specializations

Okay I thought I was pretty set on doing Machine Learning for my specialization, but I was looking them over once again today and they honestly all sound interesting...

Pros and Cons to each specialization

Computational Perception and Robotics

  • Pros:
    1. Artificial Intelligence is a core course
    2. Computational Photography and Computer Vision are Electives
    3. 3D reconstruction is an elective
    4. Multiview Geometry in Computer Vision Robotics is an elective
    5. Only 15 hours of Core courses (as opposed to 18 for others)
  • Cons:
    1. Other than Into to GA, none of the Core courses look overly interesting to me
    2. Human-Robot Interaction(?) is an elective
    3. Interactive Robot Learning(?) is an elective
  • Overall thoughts: Some of the electives don't really interest me, but there are many Camera-view based courses that I would like to take

Computing Systems

  • Pros:
    1. Advanced Operating Systems is a core course
    2. High performance Computer architecture is a core course
  • Cons:
    1. 18 hours for core and elective courses
    2. This is more so heavy on networks, computing optimizations, and software design.
  • Overall Thoughts: Not big on the concepts you would learn in this compared to what I would learn in all 3 other specializations. I am interested in them, but less so than I am with algorithms, ML, and other things. Rank Computational Perception and Robotics higher than this.

Interactive Intelligence

  • Pros:
    1. Artificial Intelligence is a core course
    2. Machine Learning is a core course
    3. 15 hours for completion
  • Cons:
    1. Human-Computer Interaction is an elective (Agile Development... lol)
  • Overall Thoughts: There is a lot of ethics involved in this specialization. Sounds like a lot of writing and not a lot of algorithms.

Machine Learning

  • Pros:
    1. Computer Vision is an elective
    2. 15 hours for completion
    3. Deep learning is an elective
    4. Machine Learning Theory is an elective
    5. Natural Language is an elective
    6. Pattern Recognition, markov chain/monte carlo is an elective
    7. A lot of good courses in the core courses
    8. Machine Learning for Trading is an elective
  • Cons:
  • Overall Thoughts:

Final Ranking

  1. Machine Learning
  2. Computational Perception and Robotics
  3. Computing Systems
  4. Interactive Intelligence

Yeah... I'm gonna do Machine Learning.
