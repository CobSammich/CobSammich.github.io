Chapter 3.5: Control Flow

This section explains how to use if statements and loops.

if Expressions

Rust has the typical if, else if, and else statements for logic control. An example of the syntax is below.

fn main() {
    let number = 6;

    if number % 4 == 0 {
        println!("number is divisible by 4");
    } else if number % 3 == 0 {
        println!("number is divisible by 3");
    } else if number % 2 == 0 {
        println!("number is divisible by 2");
    } else {
        println!("number is not divisible by 4, 3, or 2");
    }
}

Conditional statements must evaluate to booleans, otherwise an error will occur. This is different from Python where anything that isn't zero is considered True.

fn main() {
    let number = 3;

    // this expression does not evaluate to a boolean -- ERROR
    if number {
        println!("number was three");
    }
}

Using if in a let Statement

Like Python, you can assign a value based on a condition. This is because if is an expression - not a statement (See Section 3.3). Therefore we can do the following:

fn main() {
    let condition = true;
    let number = if condition { 5 } else { 6 };

    println!("The value of number is: {}", number);
}

Rust cannot handle mismatched types in the statements like above. For example, if you try to set the number variable to a str in the else expression, you will receive an error. Rust needs to know what the variable number is at compile time so it knows how to use number throughout the rest of the program.

Repetition with Loops

Rust has three kinds of loops: loop, while, and for.

Using loop

The loop keyword will execute forever until the code explicitly tells it to stop with a break statement. You can also use the continue keyword to tell the program to skip all code below it and go straight to the next iteration of the loop.

For nested loops, the break and continue keywords apply to the innermost loop. "You can optionally specify a loop label on a loop and then use the label with break or continue to have those keywords applied to the labeled loop instead of the innermost loop". Hereâ€™s an example with two nested loops:

fn main() {
    let mut count = 0;
    'counting_up: loop {
        // this loop will iterate 3 total times.
        println!("count = {}", count);
        let mut remaining = 10;

        loop {
            // this will iterate 3 more times then go back out to the outer loop
            if remaining == 7 {
                break;
            }
            if count == 2 {
                // break outer loop
                break 'counting_up;
            }
            println!("remaining = {}", remaining);
            remaining -= 1;
        }

        count += 1;
    }
    println!("End count = {}", count);
}

The outerloop here has the label 'counting_up allowing us to specify that we want to break the outer loop if count == 2

Returning Values from Loops

You can add a value to the break expression to return that value in the loop. To tell Rust we expect a value returned from a loop, we write let value = loop { then to return a value with the break statement we write break counter * 2;. The code below shows how this is done:

fn main() {
    let mut counter = 0;

    let result = loop {
        counter += 1;

        if counter == 10 {
            break counter * 2;
        }
    };

    println!("The result is {}", result);
}

Conditional Loops with while

while loops are executed while a specific condition is true. This is present in many other programming languages. The while loop can be stopped at any point using a break statement.

fn main() {
    let mut number = 3;

    while number != 0 {
        println!("{}!", number);

        number -= 1;
    }

    println!("LIFTOFF!!!");
}

Looping Through a Collection with for

You can loop over collections using a for loop like in Python using for element in collection:

fn main() {
    let a = [10, 20, 30, 40, 50];

    for element in a {
        println!("the value is: {}", element);
    }
}

The while loop above can also be written using a for loop and a Range, which is available in the standard library. A Range starts at one number and ends before another number. You can use the .rev() function to reverse a Range. Here is how the while loop could be written using a for loop:

fn main() {
    for number in (1..4).rev() {
        println!("{}!", number);
    }
    println!("LIFTOFF!!!");
}
