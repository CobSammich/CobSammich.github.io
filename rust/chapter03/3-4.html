Chapter 3.4: Comments...

Comments are done like they are in C, C++, and Java:

// Comments can be made on multiple lines. There is no way of doing multi-line
// comments like in Python, Java, C, etc...

fn main() {
    // this is a comment explaining what this program is doing
    some_function_that_does_something()
}
